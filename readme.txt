== Description ==

The Page Plugin lets you easily embed and promote any Facebook Page on your website. Just like on Facebook, 
your visitors can like and share the Page without having to leave your site. 

[fbox] use shortcode in page or post .

<?php echo do_shortcode("[fbox]") ?>


<h3>New Facebook Fan Page Widget Features :</h3>
1. Very Easy to use
2. Just install the plugin, set options in widget area
3. Comes with shorcode[fbox]



== Installation ==
Installing this plugin is pretty simple like all other wordpress plugins. Here are 2 ways to install :

<h3>How  to  Install</h3>

1. Download the zipped plugin from wordpress repository
2. Extract the zipped folder
3. Upload facebook-fan-page-widget to the /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go To Widgets page and drag Facebook Fan Page Widget to an Widget area.
6. Select the options and Save




== Frequently Asked Questions == 
= How can I Show it on my Website ? =
1. Go To Widgets page and drag Facebook Fan Page Widget to an Widget area.
2. Select the options and Save
3. The Widget will be showed on your website!
